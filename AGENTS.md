# AGENTS.md — Rhythm English

## 项目概述
- 英语发音练习应用，重点在节奏/重音训练
- 纯前端项目，HTML + CSS + JavaScript
- 部署在 GitHub Pages
- 核心功能：跟读练习 + AI 教练 + 自由练习

## 技术栈
- 原生 JavaScript
- Web Speech API（语音合成 + 语音识别）
- 可能使用外部 TTS/STT API
- CSS3 动画

## 核心功能
- 跟读练习：播放示范音频 → 用户跟读 → 评分反馈
- AI 教练：基于用户发音给出改进建议
- 自由练习：输入任何英语句子练习
- XP/天数追踪：学习进度记录

## 测试方式
- 打开 index.html，0 错误
- 测试：播放示范 → 录音跟读 → 获得反馈
- 检查麦克风权限处理
- 中英文切换
